HTML Form Elements Reference

Tag or Attribute Formatted Tag Description
Form <form> Defines an HTML form for user input.
Input <input> Allows the user to enter data (e.g., text, radio buttons, checkboxes).
Label <label> Defines a label for an <input> element.
Button <button> Creates a clickable button inside a form.
